# reCFX Go-to-Market Plan

## Target Users

### Primary — sFX Holders
Any user holding sFX from SHUI Finance who wants to earn more than base staking yield (~4% APY) without additional capital. The restaking premium targets 2-5% additional APY on top of SHUI's base, for a combined 6-9% APY. Addressable market: full SHUI Finance TVL.

### Secondary — Protocol Teams
Any team building a keeper network, oracle, or on-chain service on Conflux that needs credible decentralized security without bootstrapping their own token. 1-2 year shortcut compared to building a native validator set.

### Tertiary — dForce Lending Users
reCFX as dForce collateral enables leveraged restaking strategies, driving demand beyond pure yield-seeking.

## Distribution Strategy

### Phase 1 — Hackathon (Complete)
- Deployed on Conflux eSpace testnet with MockSFX
- Full restaking lifecycle demonstrated: deposit → delegate → earn → slash
- Submitted to Global Hackfest 2026

### Phase 2 — Mainnet Launch (Month 1-2)
- Direct partnership with SHUI Finance (mutually additive TVL)
- Apply for Conflux Ecosystem Grants Program
- Onboard 3 initial operators: Conflux keeper network, Pyth oracle relay, developer tool
- Launch dForce integration proposal for reCFX as collateral

### Phase 3 — Scale (Month 3-6)
- Expand to 10+ registered operators
- Launch reCFX/CFX liquidity pool on WallFreeX
- Introduce governance module
- Target $500K TVL within 90 days of mainnet
