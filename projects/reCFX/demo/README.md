## How to Test
1. Connect MetaMask or Fluent Wallet on Conflux eSpace Testnet
2. Get testnet CFX from [https://efaucet.confluxnetwork.org/](https://efaucet.confluxnetwork.org/)
3. Mint sFX from the Faucet page
4. Deposit sFX → receive reCFX
5. Browse operators on the Operators page
6. Delegate reCFX to an operator
7. Watch rewards accrue, claim anytime
