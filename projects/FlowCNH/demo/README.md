## How to Demo

1. Open https://flowcnhxyz.vercel.app/dashboard in two browser windows
2. Connect the employer wallet (0x09b0...10E0) in window 1
3. Connect a worker wallet in window 2
4. Create a stream from the employer window
5. Watch the worker's claimable balance tick up second-by-second
6. Click Withdraw from the worker window — gas is sponsored
