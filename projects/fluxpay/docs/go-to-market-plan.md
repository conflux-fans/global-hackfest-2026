# Go-To-Market Plan

## Positioning
FluxPay is a hosted crypto checkout for Conflux eSpace that helps merchants and developers accept stablecoin payments without building payment infrastructure from scratch.

## Primary users
- Small online merchants
- Web3 apps with subscription or one-time payment flows
- Builders who need hosted checkout links
- Teams that want webhook-based payment automation

## Core value proposition
- Faster time to launch crypto payments
- Cleaner merchant experience
- Hosted checkout + payment links
- SDK for integration
- Better usability around stablecoin payments on Conflux

## Distribution plan
1. Open-source credibility through GitHub
2. Developer onboarding through simple docs and SDK examples
3. Hackathon visibility and ecosystem exposure
4. Early pilots with Conflux-native projects and merchants
5. Social proof through demo videos, case studies, and real payment flows

## Near-term milestones
- Improve docs and quickstart
- Strengthen webhook reliability
- Improve settings, branding, and reporting
- Pilot with early users
- Collect feedback and iterate toward mainnet readiness

## Why this matters for Conflux
FluxPay can help grow:
- stablecoin utility
- merchant adoption
- developer activity
- payment-related on-chain usage
